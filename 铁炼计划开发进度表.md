# 军旅健身App "铁炼计划" 开发计划表

## 技术栈确认
- **后端**: NestJS (Node.js框架) + TypeScript
- **移动端**: UniApp + Vue 3
- **管理后台**: Ant Design Pro + React 18 + UmiJS

## 第一阶段：核心框架搭建（1周）

### 第1天-第2天：后端核心框架
1. **数据库连接设置**
   - 安装MongoDB依赖：`@nestjs/mongoose`, `mongoose`
   - 配置数据库连接模块

2. **API基础结构**
   - 创建核心模块（用户、训练、成就）
   - 设计RESTful API接口规范
   - 实现基础中间件（日志、异常处理）

### 第3天-第4天：移动端基础框架
1. **UniApp项目配置**
   - 安装必要依赖：Vuex/Pinia状态管理
   - 配置网络请求模块（封装uni.request）
   - 设计基本UI主题（军旅风格）

2. **应用结构设计**
   - 实现TabBar导航（训练、数据、我的）
   - 搭建基本页面路由

### 第5天-第7天：管理后台框架
1. **登录及权限系统**
   - 配置管理员权限系统
   - 实现登录页面

2. **基础布局**
   - 配置菜单结构
   - 实现基本CRUD组件模板
